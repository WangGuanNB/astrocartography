Install in three steps

No API keys and no tracking: install the plugin, activate it, then drop the shortcode wherever you want the experience to appear.

Install the plugin

In wp-admin, go to Plugins → Add New, search for Astro Zodiac Widget, then click Install Now.

Activate

Click Activate. The plugin adds the shortcode handler—no extra setup screens required.

Paste the shortcode

Edit any page or post (Block or Classic editor) and add [astro_zodiac]. Publish and you are done.

Shortcode examples

Use the default block title or customize it for your layout. Both work in Gutenberg shortcode blocks and the Classic editor.

Default

[astro_zodiac]

Custom heading

[astro_zodiac title="What's Your Sign?"]

Frequently asked questions

Quick answers for site owners, editors, and anyone reviewing the plugin listing.

Does the Astro Zodiac Widget collect or store visitor birthdays?
No. The experience is designed to run client-side for privacy: visitors can enjoy a zodiac sign readout without accounts, and you do not need to store their birth dates in WordPress.
Will it slow down my site?
The footprint is small and front-end oriented. As with any shortcode, avoid placing dozens of instances on a single page unless you need to—one widget per article is the typical pattern.
Does it work with the block editor and Classic?
Yes. Add a Shortcode block in Gutenberg, or paste directly in Classic. The same [astro_zodiac] syntax applies everywhere.
Is this related to astrocartography maps?
This plugin focuses on quick Western zodiac and horoscope-style copy from a birthday. For full relocation charts and line maps, visit astrocarto.org—the same team behind this Astro Zodiac Widget, bridging casual star curiosity with deeper astrocartography tools.
Is the WordPress.org download link live?
The listing becomes available at wordpress.org/plugins/astro-zodiac-widget/ once the plugin passes review. This page stays online so reviewers and early users always land on a real destination.
